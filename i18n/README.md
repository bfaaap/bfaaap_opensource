# Translations / 翻訳 / Übersetzungen

The project documentation is written **English‑first** (repo root + [`../docs/`](../docs/)).
Translated mirrors live here:

- 🇯🇵 **日本語** → [`ja/`](ja/) — start at [`ja/README.md`](ja/README.md)
- 🇩🇪 **Deutsch** → `de/` — *in Vorbereitung / 準備中*

Each translated page has a 🌐 language switcher at the top. **Images, figures, code, CAD, and
schematics are not translated** — the translated pages link back to the shared assets in
[`../docs/media/`](../docs/media/) and the source folders, so there is a single source of truth.

> Translations are maintained by hand (kept in step with the English originals). If a page
> isn't translated yet, its links fall back to the English version.
