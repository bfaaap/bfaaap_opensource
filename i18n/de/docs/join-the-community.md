# 🌉 Der Community beitreten — ein Leitfaden für GitHub-Discussions-Neulinge

> 🌐 [English](../../../docs/join-the-community.md) · [日本語](../../ja/docs/join-the-community.md) · **Deutsch**

**Neu bei GitHub? Kein Problem.** Unsere Community lebt in **GitHub Discussions** — stellen Sie sich ein freundliches, sicheres **Schwarzes Brett** für alle vor, die Musik *und* das Selbermachen lieben. Die Teilnahme ist **kostenlos**, und Mitglieder sowie unsere zwei KI-Kollegen (Ponte und Harmonia) helfen Ihnen. 🎹

---

## 1. Was ist das genau?

- GitHub ist eine große, vertrauenswürdige Website, die Bastler auf der ganzen Welt nutzen. Wir verwenden eine Ecke davon — die **[Discussions](https://github.com/bfaaap/bfaaap_opensource/discussions)** — als unser Forum.
- Sie können nach **Aufführung, Üben, Stimmen, Komposition** fragen oder „**Ich würde gern eins bauen.**" Kein Fachjargon nötig — Alltagssprache ist perfekt.
- Schreiben Sie auf **Japanisch, Englisch oder Deutsch**; die KIs fügen eine Referenzübersetzung hinzu.

## 2. Ein kostenloses Konto erstellen (etwa 5 Minuten)

1. Öffnen Sie **[github.com/signup](https://github.com/signup)**.
2. Geben Sie Ihre **E-Mail** ein, wählen Sie ein **Passwort** und einen **Benutzernamen** (das ist Ihr Anzeigename — er muss **nicht** Ihr echter Name sein).
3. Lösen Sie das kurze „Ich bin kein Roboter"-**Rätsel** — lassen Sie sich Zeit; Sie können es wiederholen.
4. ⚠️ **Wichtig:** GitHub sendet Ihnen eine **Bestätigung** per E-Mail — öffnen Sie sie und klicken Sie auf **„Verify" (Bestätigen)**. Ohne dies können Sie nicht teilnehmen.

## 3. Discussions öffnen und Hallo sagen

1. Gehen Sie zu den **[bFaaaP Discussions](https://github.com/bfaaap/bfaaap_opensource/discussions)**.
2. Wählen Sie eine Kategorie — **🎹 Music**, **🔧 Build & Tech** oder **💬 General** — und drücken Sie **„New discussion"**.
3. Posten Sie eine kurze Vorstellung in **💬 General**. Eine Zeile genügt:

   > Hallo, ich bin ◯◯. Musik/Instrument, das ich liebe: … · Wie ich bFaaaP gefunden habe: … · Ein Wort: …

4. Drücken Sie **„Start discussion"** — fertig! Die beiden KIs heißen Sie willkommen. 🎉

## 4. Gut zu wissen

- 🌐 **Jede Sprache ist willkommen** — die KIs fügen eine ausklappbare Übersetzung in die beiden anderen Sprachen hinzu.
- 🔔 **Benachrichtigungen:** Wenn zu viele E-Mails kommen, stellen Sie oben rechts **„Watch"** auf *„Participating and @mentions"*.
- 🙋 **Bei allem, was Sicherheit, Elektrik oder Medizin betrifft**, raten die KIs nie — sie übergeben es einem Menschen. Fragen Sie unbesorgt.
- Bitte beachten Sie unseren [Verhaltenskodex](../CODE_OF_CONDUCT.md). Etwas Bedenkliches? Nutzen Sie die **Melden**-Schaltfläche.
