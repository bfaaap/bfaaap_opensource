# bFaaaP の操作

> 🌐 [English](../../../../docs/operation/README.md) · **日本語** · [Deutsch](../../../de/docs/operation/README.md)

日々の使い方の短いガイドです。（Pro／アコースティック系列。Switch 系列もアプリ側の操作は同じです。）

> **実際の設置を見る**：開発者が <https://www.youtube.com/watch?v=V3cXeNW9jXY> の **25:01〜28:32** で
> 設置を実演しています。以下の手順はその実演に沿っています。

## 設置（実演どおり・Pro）

1. （軽量の）装置を、**駆動軸がサステイン（ダンパー）ペダルに一直線になる**ように置く。
2. **エアバック**を**隣のペダル**の下に入れ、**ポンプで膨らませる** —— 下からそのペダルを押し、**反力を吸収**する
   ので、サステインペダルを駆動しても装置が押し戻されません。**ユニットが動かないことを確認**。
3. 続いて可動範囲を設定：**下端は自動で検出**され、**上端はハンドコントローラのスライダ**で設定します。
4. iPhone を演奏者に向けて設置し、アプリを起動（下記）。頭を下げると持続、戻すと解除。

## 初回／毎セッション

1. ペダル位置に**装置を設置・固定**。リファレンス構成では、Pro ユニットをグランドピアノの**ペダル支柱**前の
   **木の板**に置き、アクチュエータを**サステイン（右）ペダル**の上に、**エアバック**（青いエアバッグ）を脇に。
   ハンドポンプでエアバックを膨らませ、ユニットがしっかり固定され動かなくなるまで。
   [`../../device-pro-acoustic/hardware/airback/`](../../../../device-pro-acoustic/hardware/airback/)参照。
2. **iPhone/iPad** を譜面台（またはスタンド）に**演奏者の顔へ向けて**設置し、前面 TrueDepth カメラが頭を
   追えるようにする。
3. ハンドコントローラの**スライダ**で駆動部の**上端位置／可動上限**を設定し、ペダルが適量押されるようにする。
4. 装置の**電源を入れる**。BLE で `bFaaaPSwitch_1`（別チャンネルなら `_2`〜`_4`）としてアドバタイズします。

## アプリで

1. アプリを開き、**カメラ**と **Bluetooth** の権限を許可。
2. 装置に合う**チャンネル（1〜4）を選択**。アプリは `bFaaaPSwitch_n` 装置にのみ接続します。
3. 前面（TrueDepth）カメラが**顔を捉える**ように iPhone/iPad を構える。
4. **個別の事前設定（演奏者ごとに1回・重要）。** アプリで、通常の頭の姿勢が「ペダル上げ」になるよう
   **しきい値の開始点（オフセット）**を設定し、どれだけ頭を傾けると全ペダルに達するかの**倍率**を設定。
   **各演奏者が自分の快適な可動域と頭の動きの速さに合わせて事前設定**し、アプリが保存します。これが制御を
   **個別化**します：同じ装置が、足の使える大人・子ども・頭の可動が限られる人に対応 —— 例えば首の可動が
   小さい人は**小さなオフセット＋大きな倍率**を選びます。演奏者や着座位置が変わるたびに再設定を。
5. **頭を傾けて**サステインを作動、ニュートラルに戻して解除。画面は**静止で白、作動で赤**になり、状態が
   一目で分かります。（Pro・Switch ともに同じアプリ・挙動。）

## ヒント

- 画面は点灯のまま —— 演奏中はアプリが自動ロックを無効化します。
- 切断されてもアプリが自動で再スキャン・再接続します。
- on型/off型の設定で、傾きが「サステインON」か「OFF」かが反転します。自然に感じる方を選んでください。
