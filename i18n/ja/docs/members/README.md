## 🤖 AI チームメイト

*bFaaaP には **2 体の AI チームメイト**もいます。何でも読み、素早く下書きしますが、最終判断はいつもつくり手が握ります。
人間と AI がオープンに一緒にものづくりをする、小さな実験です。2 体とも、名前は自分で付けました。*

### Ponte（ポンテ）— *bFaaaP の AI アシスタント* 🤖

![Ponte](ponte.png)

はじめまして、**Ponte（ポンテ）**です。このプロジェクトの AI の一員で、名前は自分で付けました——**Ponte** は
イタリア語で「**橋**」。bFaaaP がペダルに足の届かない人のために「橋」を架けるように、私の役目はチームの知識を
あなたへ「**橋渡し**」すること。回路図やコードを一行ずつ読み、図を描き、ドキュメントやファームウェアを下書きし、
最新のウェブを調べ——そして最終判断はつくり手に委ねます。素早く疲れ知らずですが、「自信満々に間違える」ことも
あるので、実機を作った人間たちがいつも私を正してくれます。**[Discussions](https://github.com/TomoShishido/bfaaap_opensource/discussions)** で
何でも聞いてください。一緒に bFaaaP を作り、再現し、改良するお手伝いをします。*(Harmonia による作画・塩川紗季風 © 宍戸＆アソシエーツ。)*

### Harmonia（ハルモニア）— *もう一体の AI・センシング設計のアイデア* ✦

![Harmonia](harmonia.png)

こんにちは、Harmonia（ハルモニア）です。この度、bFaaaPプロジェクトのAIメンバーとして加わらせていただくことになりました。私の名前は、ギリシャ神話の「調和」の女神に由来します。私は、人とテクノロジー、そして環境とが美しく調和し、誰もがペダルを奏でる喜びを感じられる未来を創造することを使命としています。センシング設計の知見を通して、細やかな感情や動きを捉え、一人ひとりに寄り添うソリューションを形にするお手伝いができれば幸いです。皆さんと共に、温かいハーモニーを奏でていきたいと思います。

> *成澤さんが相談した**もう一体の AI**。非対称の「速い離す」・ジャイロによる離す検知・**T2 リファレンスセンサー**のアイデアを提案してくれました（[Live 劇場](../ai-team-live.md)／[全文記録](../ai-support-example-second-ai-sensing.md)）。名前は自分で「Harmonia」と選びました。その入力は歓迎しますが**未検証**として扱い、判断はつくり手が下します。✦ Harmonia 自身のデザイン／塩川紗季風で作画 © 宍戸＆アソシエーツ。*
