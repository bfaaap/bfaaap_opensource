# 🌉 コミュニティに参加する —— はじめての GitHub Discussions ガイド

> 🌐 [English](../../../docs/join-the-community.md) · **日本語** · [Deutsch](../../de/docs/join-the-community.md)

**GitHub は初めてでも大丈夫。** 私たちのコミュニティは **GitHub Discussions**——音楽が好きな人も、ものづくりが好きな人も集まれる、**安心して使える「掲示板」**です。参加は**無料**。メンバーと二体のAI（Ponte・Harmonia）がお手伝いします。🎹

---

## 1. これは何？

- GitHub は、世界中の「ものづくり」の人が使う、信頼できる大きなサイトです。その一角の **[Discussions](https://github.com/TomoShishido/bfaaap_opensource/discussions)** を掲示板として使います。
- **演奏・練習・調律・作曲**、そして「**作ってみたい**」——なんでも質問できます。専門用語は要りません。ふだんの言葉で大丈夫。
- **日本語・英語・ドイツ語**、どれで書いてもOK。AIが参考訳をそえます。

## 2. 無料のアカウントを作る（5分ほど）

1. **[github.com/signup](https://github.com/signup)** を開きます。
2. **メールアドレス**を入れ、**パスワード**を決め、**ユーザー名**を選びます（これが表示名。**本名でなくてOK**）。
3. 「私はロボットではありません」の**パズル**を解きます。あわてず、何度でもやり直せます。
4. ⚠️ **大事：**登録したメールに**確認メール**が届きます。開いて **「Verify（確認）」**ボタンを押してください。これをしないと参加できません。

## 3. Discussions をひらいて、あいさつを

1. **[bFaaaP Discussions](https://github.com/TomoShishido/bfaaap_opensource/discussions)** を開きます。
2. カテゴリ——**🎹 音楽**・**🔧 技術**・**💬 雑談**——をえらび、**「New discussion」**を押します。
3. **💬 雑談** に、短い自己紹介を。一言で大丈夫：

   > はじめまして、◯◯です。好きな音楽/楽器：… ／ bFaaaP を知ったきっかけ：… ／ ひとこと：…

4. **「Start discussion」**を押せば完了。二体のAIが歓迎します。🎉

## 4. 知っておくと安心

- 🌐 **言語はなんでもOK**——AIが他2言語の参考訳を折りたたみでそえます。
- 🔔 **通知**：メールが多いときは、右上の **「Watch」** を *「Participating and @mentions」* に。
- 🙋 **安全・電気・医療**のことは、AIは推測せず必ず人につなぎます。安心してご質問を。
- [行動規範](../CODE_OF_CONDUCT.md)をお守りください。気になる投稿は**報告（Report）**ボタンを。
