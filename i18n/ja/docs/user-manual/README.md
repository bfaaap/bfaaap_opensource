# bFaaaP —— 取扱説明（アプリ＋装置）

> 🌐 [English](../../../../docs/user-manual/README.md) · **日本語** · [Deutsch](../../../de/docs/user-manual/README.md)

プロジェクト公開マニュアル（<https://scrapbox.io/bfaaap/switch>）をもとにしています。これは**公開版 Switch**
マニュアルです。Pro 装置もアプリ側の操作は同じです（Pro の物理設置は
[デモ動画](https://www.youtube.com/watch?v=V3cXeNW9jXY)と[`../operation/`](../operation/)を参照）。

> **マニュアル動画：** <https://youtu.be/XOVENtBsOp4>

## 1. これは何か

**電子ピアノ／キーボード**のペダルを、iOS（または Android）アプリの AI で演奏者の**頭の角度**を検出して
操作します —— 足を使わずに音を持続（「歌わせる」）。bFaaaP Switch には **Switch 装置とアプリの両方**が必要です。
Switch 装置（**技適 R018‑180280**）は**受注生産** —— まず <https://ui.saaipf.com/app3/> で購入し、その後にアプリを**無料でダウンロード**します。

## 2. 装置を電子ピアノに接続

1. 装置のケーブル端を確認 —— **6.3mm 2極（TS）フォーンプラグ**。
2. ピアノの**サステインペダル端子**（6.3mm 2極）を確認。
3. ケーブルを **SUSTAIN** ジャックに挿す。*（3ペダルの「PEDAL UNIT」は配線が異なるので注意。）*

## 3. 装置とアプリの電源

- **装置上部のボタン**を押す → **青く点滅**。
- iOS アプリを開く。

## 4. BLE チャンネルを選択

- チャンネルボタンは接続可能になるまで灰色。**青くなったら**タップ。

## 5. 開始

- **初回**は頭を上下に振り、画面が**白 ↔ 赤**に切り替わるようにする —— これで角度検出が装置に引き渡されます。
  *（最初の1サイクルは作動しません —— 仕様です。）*

## 6. on型／off型

- 電子ピアノによってサステインの論理が反転します（頭を下げる→赤→サステインが逆になることがある）。最初に
  自動切替されることも。必要なら **On/Off ボタン**で反転してください。

## 7. 頭部角度のしきい値

- 選んだニュートラル位置で頭を保ち、**Angle Set** を押してしきい値を固定。
- しきい値未満：画面**灰色**、装置 LED **緑**。
- しきい値以上：画面**赤**、装置 LED **赤**、音が持続。

## 8. サポートページ

Support ボタンから、このマニュアル・**健康と安全の注意**・**プライバシーポリシー**
（<https://bfaaap.com/bfaaapについて/privacy-policy/>）・**EULA** にリンクします。

## 9. BLE チャンネルの変更

- bFaaaP Switch を**2台以上**使うときはチャンネルを変更。**Reset** を押すとチャンネルが変わり装置が再初期化されます。
