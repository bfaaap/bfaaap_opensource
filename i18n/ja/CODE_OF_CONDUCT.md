# 行動規範 — bFaaaP コミュニティ

> 🌐 [English](../../CODE_OF_CONDUCT.md) · **日本語** · [Deutsch](../de/CODE_OF_CONDUCT.md)

**技術と音楽が出会う場所へ、ようこそ。**

bFaaaP は、開かれた、インクルーシブなプロジェクトです。足を使わずに演奏できるピアノペダルで、誰もがピアノを楽しめるように。私たちのコミュニティ——**[GitHub Discussions](https://github.com/TomoShishido/bfaaap_opensource/discussions)**——は、技術**と**音楽を愛するすべての人が、安心して知識を交換し、創意を分かち合い、互いに高め合える場所です。これは**支援技術（assistive technology）**であり、共感（エンパシー）が私たちの土台です。以下のガイドラインを守り、よいコミュニティを一緒に育てていきましょう。

## 私たちの約束

1. **敬意をもって。** 経歴・障害の有無・経験の度合いにかかわらず、あらゆる人を歓迎します。個人攻撃・ハラスメント・差別、そして初心者を見下すことは、いずれも認められません。
2. **建設的に。** やさしく答えましょう。批判するなら、人ではなくアイデアに対して。誰もが学び成長できる対話を目指します。基本的な質問はいつでも歓迎です——私たちも皆、最初は初心者でした。
3. **安全に。** 著作権やプライバシーを侵害するもの、不適切なものは投稿しないでください。気になる投稿を見かけたら、**報告（Report）**ボタンをお使いください。
4. **カテゴリを守って。** 投稿は適切なカテゴリへ——**🔧 技術 / Build & Tech**・**🎹 音楽 / Music**・**💬 雑談 / General**——公式のお知らせは **📣 Announcements** へ。ご自身のレッスン・調律・演奏会の紹介は、**💬 雑談 / General** や自己紹介で、節度をもってどうぞ。ただし、回答を宣伝の場に変えるのはご遠慮ください。

## ここでは歓迎されないもの

スパムや広告、フィッシング／マルウェアのリンク、話題から外れた投稿、敵対的・非礼なふるまい、ハラスメント、個人情報の暴露（ドクシング）、そして**自動化された、または大量の投稿**（ボットやAIスクレイパーによる一斉投稿）は、非表示または削除の対象となり、返信もされません。

## 音楽カテゴリについて

- **初心者を歓迎します。** どんなに基本的なことでも、なんでも質問してください。経験者のみなさんは、どうか寛大に、やさしく。
- **著作権と楽譜。** 著作権のある楽譜・歌詞・音源を、権利者の許可なく（全体でも一部でも）アップロードしないでください。合法的な、あるいはパブリックドメインの資料（例：IMSLP）をお使いください。
- **演奏動画。** 動画に他の人が映っている場合は、事前に本人の許可を得てください。カバー演奏では、その曲の著作権にご注意を。
- **宣伝。** レッスン・調律サービス・演奏会の紹介は歓迎します——**💬 雑談 / General** や自己紹介で、節度をもって。回答を宣伝の場に変えるのはご遠慮ください。

## どの言語でも歓迎

**日本語・英語・ドイツ語**——ご自身が書きやすい言語で投稿してください。Ponte と Harmonia が、あなたの投稿の下に、残りの2言語への参考訳を折りたたみで添えます。みんなが読み進められるように。**原文が正であり、AIによる翻訳はあくまで補助**です——もし訳がおかしければ、`@Ponte @Harmonia` とお知らせください。修正します。

## モデレーションの方針（人が主役）

このコミュニティは生身の人間が運営し、**最終判断は人が下します**——それが、大切なこと（配線、24V電源、電波法、支援機器の安全）についての答えを信頼できるものに保つ鍵です。

- **オーナー**（宍戸・田口・田中）が最終決定者です。モデレーターの任免、重大な違反への対応、方針の決定を行います。
- **モデレーター**は、日々の巡回、スレッドの整理、軽微な対応を担う、信頼されたメンバーです。
- 無理のない**月単位の当番制**（四六時中の拘束はしません）でモデレーションを安定させ、日々の巡回はAIアシスタントが担うことで、人の負担をやさしく保ちます。
- **手順：** 誰でも**報告（Report）**できます → 当番のモデレーターが、明らかなスパム・違反にはすぐ対応します（コメントは削除ではなく、短い理由を添えて**非表示**に。過熱した、または解決したスレッドは**ロック**することがあります）→ 判断に迷う場合はモデレーター間で相談します → 重大なケース（ユーザーのブロック等）はオーナーへエスカレーションします。
- **AIアシスタント**（Ponte と Harmonia）はチームの一員です。新しい人を迎え、会話を続ける手助けをし、投稿を翻訳し、危険な投稿にフラグを立て、長いスレッドを要約し、人がレビューするための返信を**下書き**します。すべての投稿に署名し、専門家の確認が必要な情報にはその旨を注記します。**安全・医療・電気・権利や法律・未確定のプロジェクト方針**にかかわることは、いずれも人に委ねます——そして**AIが自らの判断で削除・非表示・ブロックを行うことは、決してありません。最終的なボタンは、必ず人が押します。**

## 違反への対応

これらのガイドラインを著しく破るふるまいに対しては、モデレーターが投稿を非表示・削除し、コミュニティの安全を守るために、一時的または恒久的にアカウントを停止することがあります。私たちは**歓迎的でありながら毅然と**あることを目指し——投稿が単に不明確なだけなら、**拒むのではなく、まず尋ねる**ことを心がけます。

## 適用範囲と連絡先

これは bFaaaP のすべてのコミュニティの場（Discussions・issues・pull requests）に適用されます。懸念事項を非公開で報告するには、オーナー <shishid@saaipf.com> までご連絡ください。

これは行動規範の**バージョン1**です——広く使われているオープンソースの行動規範（[ベルリン行動規範](https://berlincodeofconduct.org/)・Contributor Covenant）を土台に、私たちの音楽と技術のコミュニティ向けに調整したもので、モデレーターとともにこれからも育てていきます。

**最高のコミュニティを、みんなで一緒につくりましょう。** 🌉🎹
