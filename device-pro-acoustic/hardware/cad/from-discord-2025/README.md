# CAD imported from the team Discord (2025)

These **FreeCAD (`.FCStd`) source files** are additional Pro‑device parts shared by
**Hiroyuki Narusawa** (and collaborators) in the team's private **Discord**
(`bFaaaP`) during 2025. They complement the curated set in the parent
[`../`](../) folder (airbox variants, `auto_box`, `base_plus_airjack`,
`belt_gotai14`, `dipsw_box`, hand‑controller knobs/slide, side panels, pump test,
`valp`, `cone_panel`, etc.).

The matching **new‑housing STL** (heat‑countermeasure revision, 2025‑05:
`bFaaaP_AA / CC / DD / E_a / FF`) is in
[`../../3d-print/from-discord-2025/`](../../3d-print/from-discord-2025/).

> ⚠️ **Provenance & consent.** Source: team Discord, by H. Narusawa et al.
> **Author consent + a license are required before public release** (tracked in
> [`../../../../PUBLISHING-CHECKLIST.md`](../../../../PUBLISHING-CHECKLIST.md)).
> Some files are design variants/iterations; treat the parent `cad/` set as the
> curated reference and these as the fuller as‑shared archive.

Print notes from the Discord thread: some parts are **~240 mm long** and exceed a
Bambu **A1 mini** bed (`CC`, `DD` overflowed); a larger printer
(e.g. Bambu A1 / Raise3D Pro3) is needed for those. Holes have **no support**;
tolerances are intentionally loose (~0.1–0.2 mm). See
[`../../../DISCORD-FINDINGS.md`](../../../DISCORD-FINDINGS.md) for the full log.
