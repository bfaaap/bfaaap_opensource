# CAD sources (FreeCAD)

Editable mechanical design files in **FreeCAD** format (`.FCStd`). 21 parts/assemblies.

- Open with [FreeCAD](https://www.freecad.org/) (free, open source).
- These are the source of truth; the printable meshes in `../3d-print/` are
  exported from here.
